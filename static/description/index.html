Dynamic Field Translations

Expose the EN / EL toggle on any existing field.

Administrators can opt-in specific Char, Text, HTML, or Selection fields to become translatable without patching source modules. The add-on keeps track of the original setting, so disabling a rule reverts the field gracefully.

Highlights

  • Instantly enables inline translation widgets for configured fields.
  • Respects original metadata and rolls back cleanly on uninstall.
  • Applies automatically on server restart thanks to registry hooks.

Workflow

  1. Select the target model and field.
  2. Keep the rule active to expose language toggles.
  3. Archive to temporarily disable while preserving configuration.
